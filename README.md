# 💚 Finanzas Cami & Juan

App personal de control financiero — gastos, ingresos, transferencias y ahorros.

## 🔗 Usar la app
Una vez publicada en GitHub Pages, la URL va a ser:
`https://TU_USUARIO.github.io/finanzas-app`

## 📱 Instalar en el celular
**iPhone:** Safari → Compartir → Añadir a pantalla de inicio  
**Android:** Chrome → ⋮ → Añadir a pantalla de inicio

## ⚙️ Configuración
Al abrir la app por primera vez, ir a ⚙️ Config y pegar la URL del Apps Script de Google.

## 📂 Archivos
- `index.html` — La app completa
- `manifest.json` — Configuración PWA
- `sw.js` — Service Worker (funciona offline)
- `icon-192.png` / `icon-512.png` — Íconos de la app
